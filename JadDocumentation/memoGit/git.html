Mémo GIT

Documentation officiel

Installation :

Il faut installer GIT comme décrit dans la procédure d'installation.
Pour configurer pour notre repository GITHUB tapez les commandes suivantes :


        git config --global user.name "lain076"
        git config --global user.email "djpcacc@gmail.com"
NB: Global veut dire que l'email et l'user sont globaux à tous les repos. Il ne doivent pas être redéfinit pour chaque repo.

Copilot: Je possède déjà un compte sur GitHub avec trois repositories dessus. Je vais donc le cloner sur ma machine local avec la commande "git clone". Est ce que je dois quand même faire le git config après l instalation pour l email et le nom d utimisateur?
Oui, vous devez quand même configurer votre nom et votre adresse email avec git config, même si vous avez déjà un compte GitHub et que vous clonez des dépôts existants.

🧾 Pourquoi faire git config après l’installation ?
Ces informations sont utilisées pour signer vos commits localement.

Elles permettent d’identifier qui a fait quoi dans l’historique du projet.

Même si vous clonez un dépôt existant, dès que vous commencerez à faire des modifications et des commits, Git utilisera ces paramètres.


Copilot: Est ce que je dois faire un git init dans le répertoire de destination si je clone mon dépôt dedant?
Non, vous n’avez pas besoin de faire git init dans le répertoire de destination si vous utilisez git clone.


Copilot: j ai trois dépots différents de dois donc les cloner tous les trois dans un répertoire dédié?
Tu n’es pas obligé de créer un répertoire dédié, mais c’est fortement recommandé pour garder ton environnement de travail bien organisé.

🗂️ Option recommandée : un dossier parent pour tous tes dépôts
Tu peux créer un dossier principal (par exemple MesProjetsGit) et y cloner tes trois dépôts :

bash
mkdir MesProjetsGit
cd MesProjetsGit

git clone https://github.com/ton-utilisateur/depot1.git
git clone https://github.com/ton-utilisateur/depot2.git
git clone https://github.com/ton-utilisateur/depot3.git
Cela te donnera :

Code
MesProjetsGit/
├── depot1/
├── depot2/
└── depot3/
✅ Avantages :
Tu gardes tous tes projets GitHub au même endroit.

Tu peux facilement naviguer entre eux.

Tu évites de mélanger des fichiers dans un répertoire non versionné.
